エラーハンドリング
==================

API エラー処理について説明します。

エラーレスポンス形式
--------------------

APIエラーは以下の形式で返されます:

.. code-block:: json

   {
     "error": {
       "code": "ERROR_CODE",
       "message": "エラーメッセージ",
       "details": {}
     }
   }

HTTP ステータスコード
======================

主要なHTTPステータスコード:

.. list-table:: HTTPステータスコード
   :header-rows: 1
   :widths: 20 80

   * - コード
     - 説明
   * - 200
     - 正常処理
   * - 400
     - リクエストエラー
   * - 422
     - バリデーションエラー  
   * - 500
     - サーバーエラー
