# 📦 Release MVP 機能一覧

## 🎯 目的

最小限の機能でリリースし、ユーザーに触ってもらうことを優先する。
複雑な機能や最適化は後回しにし、**動く・使える状態** を最速で公開する。

---

## ✅ MVPに含める機能

### フロントエンド

* シンプルなチャットUI

  * ユーザーが質問を入力できる
  * AIの応答を表示する
* 最低限のデザイン（見やすい程度で十分）
* デプロイ: Firebase Hosting

### バックエンド

* ユーザー入力を受け取るAPI
* OpenAI API への問い合わせ処理
* **ベクトル検索（Upstash Vector）のみ**

  * 質問に関連するカード情報を検索
  * 検索結果をプロンプトに組み込み、回答を生成
* デプロイ: Cloud Run

### データ

* カードデータ（ベクトル化済み）
* ベクトル検索に必要な最小限の情報のみ

---

## 🚫 MVPから除外する機能

（将来の拡張に回す）

* 挨拶検出・早期応答システム
* ハイブリッド検索（BM25 + ベクトル検索）
* 検索モードの自動選択（FILTERABLE / SEMANTIC / HYBRID）
* 戦略的推薦機能（例: 水タイプ＋ダメージ40以上）
* 複合条件検索
* カード詳細ページ（`/cards/{card_id}/details`）
* 高度なエラーハンドリング・動的閾値調整
* 包括的テスト（Smoke Testのみ残す）
* モニタリング（Sentry, Prometheus など）

---

## 🚀 リリース後の流れ

1. MVPをユーザーに触ってもらう
2. フィードバックを収集する
3. 必要とされた機能から順に追加していく
